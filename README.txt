# Resume Classification NLP Project


Overview:
This project involves the classification of resumes into different categories
using Natural Language Processing (NLP) techniques. The project covers data loading, 
data cleaning, exploratory data analysis (EDA), vectorization, model training, and evaluation.

Python Libraries Required :
. Pandas
. NumPy
. Matplotlib
. Seaborn
. Scikit-learn

This project demonstrates the process of resume classification using NLP techniques.
It covers data loading, cleaning, EDA, vectorization, model training, and evaluation, 
providing a comprehensive workflow for text classification tasks.
